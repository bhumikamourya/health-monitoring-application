# Health Monitoring App

A full-stack web application to track daily health metrics like water intake, calories, sleep, and weight, steps, heartRate, calories, bloodPressure, etc.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- DataBase: MongoDB

##Features
- User Authentication
- Manual health tracking
- Dashboard with charts
- Weekly Summary

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
